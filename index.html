Karnataka Dashboards 2023


Tap Water Connection Dashboard :
  • Using dropdown box where users can choose a specific district and then further refine the data by selecting a particular block within that district.
  • Hover the cursor over the map to see the details of the district with tap connection
  • The bar chart displays a comparison of the number of households with tap water connections across different blocks.
      • Coverage= (Number of households with Tap Connection/Total no. of Households)*100

    Water Quality Status Dashboard 2022-2023:
    
      
    
    • The water potability status chart represents:
        
      • X-axis: Status (Compliant, Partially Compliant, Non-compliant,Not Available).
        • 
            
          
                
        • Compliant : Samples collected are under permissible limit
          • 
                
        • Partially Compliant: all parameters are not measured, but the measured parameters are within the permissible range.
          • 
                
        • Not Compliant: At least one sample value that is outside the permissible limit.
          • 
                
        • Not Available: The villages where the sample has not been collected.
          • 
            
        
        
      • Y-axis: Village count represented in percentage.
      
    
      • 
    
    • The second page of the dashboard describes the chemical and physical parameters measured and no.of samples tested by the labs. 
      • 
    
    • The first dropdown box enables users to select and filter data based on district names within the specified region, 
        while the second dropdown allows users to filter data for a specific year
      • 
    
    • The user can turn the plot on/off by clicking on the legends.
      • 
    
    • The line chart represents chemical parameters which vary across districts over the selected year. Hover the cursor over the points 
        where it displays the average values of chemical parameters.
      • 
    
    • The bar graph represents comparision of levels of physical parameters among districts.
      • 
    
    • The upper left bar graph shows the number of samples tested by different labs and the constant line in the graph represents the benchmark which is 3000 samples
